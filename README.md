# Code-Folders
Simple application for managing project folders

### MVP features:
- [ ] browse to folder to add it to the project folders list
- [ ] display folders in listview
- [ ] click "open" button to open the selected folder in explorer
- [ ] CTRL + CLICK to open folder directly
- [ ] click "delete" button to delete the selected folder
- [ ] save the list of folders on exit
- [ ] load the list of folders on startup
